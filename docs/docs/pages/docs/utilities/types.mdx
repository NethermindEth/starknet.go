# Type Utilities

The type utilities submodule provides functionality for working with Starknet types.

## Type Conversion

### To Big Int

```go
bigInt := utils.ToBigInt(value)
```

### From Big Int

```go
value := utils.FromBigInt(bigInt)
```

### To Felt

```go
felt := utils.ToFelt(value)
```

### From Felt

```go
value := utils.FromFelt(felt)
```

## Type Validation

### Is Valid Big Int

```go
isValid := utils.IsValidBigInt(value)
```

### Is Valid Felt

```go
isValid := utils.IsValidFelt(value)
```

### Is Valid Uint256

```go
isValid := utils.IsValidUint256(value)
```

## Type Operations

### Add Big Ints

```go
result := utils.AddBigInts(a, b)
```

### Sub Big Ints

```go
result := utils.SubBigInts(a, b)
```

### Mul Big Ints

```go
result := utils.MulBigInts(a, b)
```

### Div Big Ints

```go
result := utils.DivBigInts(a, b)
```

## Example Usage

```go
// Convert to big int
bigInt := utils.ToBigInt("123")

// Convert from big int
value := utils.FromBigInt(bigInt)

// Convert to felt
felt := utils.ToFelt("123")

// Convert from felt
value = utils.FromFelt(felt)

// Validate big int
isValid := utils.IsValidBigInt("123")

// Validate felt
isValid = utils.IsValidFelt("123")

// Validate uint256
isValid = utils.IsValidUint256("123")

// Add big ints
result := utils.AddBigInts("123", "456")

// Sub big ints
result = utils.SubBigInts("123", "456")

// Mul big ints
result = utils.MulBigInts("123", "456")

// Div big ints
result = utils.DivBigInts("123", "456")
```
