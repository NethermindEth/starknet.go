# starknet_getCompiledCasm

Returns the compiled Cairo Assembly (CASM) code for a given contract class.

## Parameters

1. `contract_class` - The contract class to compile.

## Returns

An object containing:
- `compiled_class_hash` - The hash of the compiled class.
- `casm` - The compiled Cairo Assembly code, including:
  - `bytecode` - The bytecode of the compiled contract.
  - `entry_points_by_type` - Entry points organized by type.
  - `hints` - Hints used in the compilation.
  - `compiler_version` - The version of the compiler used.
