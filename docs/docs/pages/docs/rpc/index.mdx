# RPC Package

The RPC package provides complete access to Starknet's JSON-RPC API. It implements the Starknet RPC v0.9.0 specification for interacting with Starknet nodes.

## What You Can Do

**Query Blockchain Data**
- Get latest blocks and transactions
- Read contract storage and state
- Query historical events

**Submit Transactions**
- Deploy accounts and contracts
- Execute contract functions
- Estimate transaction fees

**Real-time Updates**
- Subscribe to new blocks
- Monitor contract events
- Track transaction status

## Getting Started

```go
package main

import (
    "context"
    "fmt"
    "log"

    "github.com/NethermindEth/starknet.go/rpc"
)

func main() {
    // Create RPC client
    client, err := rpc.NewProvider("https://starknet-sepolia.public.blastapi.io/rpc/v0_8")
    if err != nil {
        log.Fatal(err)
    }

    // Make RPC calls
    blockNumber, err := client.BlockNumber(context.Background())
    if err != nil {
        log.Fatal(err)
    }

    fmt.Printf("Latest block number: %d\n", blockNumber)
}
```

## Core Components

**Provider**: HTTP client for standard RPC calls
**WebSocketProvider**: Real-time subscriptions and updates
**Types**: Complete type definitions for all operations

## Method Categories

### Blocks
Methods for retrieving blockchain data at different levels of detail:

- `BlockNumber()` - Get the latest block number
- `BlockHashAndNumber()` - Get both hash and number of latest block
- `BlockWithTxHashes()` - Get block with transaction hashes only (lightweight)
- `BlockWithTxs()` - Get block with full transaction details
- `BlockWithReceipts()` - Get block with transactions and execution receipts
- `BlockTransactionCount()` - Get transaction count in a block
- `StateUpdate()` - Get state changes in a block

**Choosing the Right Block Method:**
- Use `BlockWithTxHashes()` for quick block overview with minimal data
- Use `BlockWithTxs()` when you need complete transaction details
- Use `BlockWithReceipts()` when you need execution results, events, and gas usage
- Use `BlockTransactionCount()` for efficient monitoring without downloading block data

### Transactions
`TransactionByHash()`, `TransactionByBlockIdAndIndex()`, `TransactionReceipt()`, `GetTransactionStatus()`, `AddInvokeTransaction()`, `AddDeclareTransaction()`, `AddDeployAccountTransaction()`, `GetMessagesStatus()`

### Contracts
`Call()`, `Class()`, `ClassAt()`, `ClassHashAt()`, `StorageAt()`, `Nonce()`, `EstimateFee()`, `EstimateMessageFee()`, `GetStorageProof()`, `CompiledCasm()`

### Chain
`ChainID()`, `Syncing()`, `SpecVersion()`

### Events
`Events()` with comprehensive filtering support

### Debug & Tracing
`TraceTransaction()`, `TraceBlockTransactions()`, `SimulateTransactions()`

## Network Support

Works with all Starknet networks - mainnet, sepolia testnet, and local devnets. Simply change the RPC URL to switch networks.
