# ABI Package

The ABI package provides functionality for working with Starknet Application Binary Interface (ABI). This package contains various components and utilities for encoding and decoding Starknet contract data.

## Overview

The ABI package includes:
- ABI encoding
- ABI decoding
- ABI utilities

## Available Submodules

- [ABI Encoding](./encoding)
- [ABI Decoding](./decoding)

## Getting Started

To use the ABI package, import it in your Go code:

```go
import "github.com/NethermindEth/starknet.go/abi"
```

## Version

This documentation is based on version v0.8.1 of the Starknet.go library.
