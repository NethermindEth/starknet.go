# ABI Encoding

The ABI encoding submodule provides functionality for encoding Starknet contract data.

## Encode Function Call

```go
encoded, err := abi.EncodeFunctionCall(functionName, args)
```

## Encode Constructor

```go
encoded, err := abi.EncodeConstructor(args)
```

## Encode Event

```go
encoded, err := abi.EncodeEvent(eventName, args)
```

## Encode Struct

```go
encoded, err := abi.EncodeStruct(structName, fields)
```

## Encode Array

```go
encoded, err := abi.EncodeArray(elementType, elements)
```

## Example Usage

```go
// Encode a function call
encoded, err := abi.EncodeFunctionCall("transfer", []interface{}{
    "0x123...", // recipient
    "1000",     // amount
})
if err != nil {
    log.Fatal(err)
}

// Encode a constructor
encoded, err = abi.EncodeConstructor([]interface{}{
    "MyToken",  // name
    "MTK",      // symbol
    "18",       // decimals
})
if err != nil {
    log.Fatal(err)
}

// Encode an event
encoded, err = abi.EncodeEvent("Transfer", []interface{}{
    "0x123...", // from
    "0x456...", // to
    "1000",     // value
})
if err != nil {
    log.Fatal(err)
}

// Encode a struct
encoded, err = abi.EncodeStruct("Point", map[string]interface{}{
    "x": "100",
    "y": "200",
})
if err != nil {
    log.Fatal(err)
}

// Encode an array
encoded, err = abi.EncodeArray("felt", []interface{}{
    "1", "2", "3", "4", "5",
})
if err != nil {
    log.Fatal(err)
}
```
