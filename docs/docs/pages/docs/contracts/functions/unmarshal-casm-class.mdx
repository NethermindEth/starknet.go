# UnmarshalCasmClass

Loads and unmarshals a CASM (Cairo Assembly) class from a JSON file.

## Function Signature

```go
func UnmarshalCasmClass(filePath string) (*CasmClass, error)
```

**Source:** [contracts.go:L18-L31](https://github.com/NethermindEth/starknet.go/blob/main/contracts/contracts.go#L18-L31)

## Parameters

- `filePath` (string): Path to the .casm.json file

## Returns

- `*CasmClass`: Pointer to the unmarshaled CASM class
- `error`: Error if file reading or unmarshaling fails

## Usage Example

```go
package main

import (
	"fmt"
	"log"

	"github.com/NethermindEth/starknet.go/contracts"
)

func main() {
	// Path to the compiled CASM file (generated by Scarb build process)
	casmPath := "counter_contract/target/dev/counter_Counter.compiled_contract_class.json"

	// Load and unmarshal the CASM class from file
	casmClass, err := contracts.UnmarshalCasmClass(casmPath)
	if err != nil {
		log.Fatalf("Failed to unmarshal CASM class: %v", err)
	}

	// Display CASM class information
	fmt.Printf("Compiler Version: %s\n", casmClass.CompilerVersion)
	fmt.Printf("Bytecode Length: %d instructions\n", len(casmClass.ByteCode))
	fmt.Printf("Entry Points: %d external functions\n", len(casmClass.EntryPointsByType.External))

	// Display entry point details
	for _, ep := range casmClass.EntryPointsByType.External {
		fmt.Printf("  Selector: %s, Offset: %d\n", ep.Selector.String(), ep.Offset)
	}
}
```

## Common Use Cases

- **Contract Deployment**: Load CASM classes for declare V2/V3 transactions, which require both Sierra and CASM formats. See [Loading CASM Classes](/docs/contracts/examples#loading-casm-classes).
- **Bytecode Analysis**: Inspect the compiled bytecode, entry points, and compiler version to understand contract structure. See [Working with Entry Points](/docs/contracts/examples#working-with-entry-points).
- **Class Hash Calculation**: Use the loaded CASM class to compute the compiled class hash needed for contract declaration.
- CASM files are generated by the Cairo compiler and contain the executable bytecode that runs on the Starknet VM.
