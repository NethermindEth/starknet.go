# ContractClass (Type)

Represents a Sierra contract class in Starknet.

## Type Definition

```go
type ContractClass struct {
    SierraProgram        []*felt.Felt
    ContractClassVersion string
    EntryPointsByType    SierraEntryPointsByType
    ABI                  NestedString
}
```

## Fields

- `SierraProgram` - List of Sierra instructions
- `ContractClassVersion` - Version (e.g., "0.1.0")
- `EntryPointsByType` - Entry points by type (constructor, external, L1 handler)
- `ABI` - Contract ABI in JSON format

## Usage

```go
var contractClass contracts.ContractClass
json.Unmarshal(data, &contractClass)

classHash := hash.ClassHash(&contractClass)
```
