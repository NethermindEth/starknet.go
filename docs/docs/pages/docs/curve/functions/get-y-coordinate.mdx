# GetYCoordinate

Returns the Y-coordinate given an X-coordinate on the Stark curve. This function solves the curve equation to derive the Y-coordinate from a known X-coordinate.

## Function Signature

```go
func GetYCoordinate(starkX *felt.Felt) *felt.Felt
```

**Source:** [curve.go:L170-L193](https://github.com/NethermindEth/starknet.go/blob/main/curve/curve.go#L170-L193)

## Parameters

- `starkX` (*felt.Felt): The x-coordinate of the point

## Returns

- `*felt.Felt`: The y-coordinate of the point

## Usage Example

```go
package main

import (
	"fmt"

	"github.com/NethermindEth/juno/core/felt"
	"github.com/NethermindEth/starknet.go/curve"
)

func main() {
	// Known X-coordinate
	x, _ := new(felt.Felt).SetString("0x5f1614a304f6a40630792a4f6562eee36805f52e94f777d6a78463ae541229b")

	// Derive Y-coordinate
	y := curve.GetYCoordinate(x)

	fmt.Println("GetYCoordinate:")
	fmt.Printf("  X-Coordinate: %s\n", x.String())
	fmt.Printf("  Y-Coordinate: %s\n", y.String())
}
```

## Common Use Cases

- Reconstruct complete public key points when only the X-coordinate is available from storage or network data.
- Verify that a given X-coordinate corresponds to a valid point on the Stark curve.
- Use in compressed key representations where Y-coordinates are omitted to save space.
- Complete elliptic curve operations that require both coordinates when working with Starknet addresses.
