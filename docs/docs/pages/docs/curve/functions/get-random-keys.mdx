# GetRandomKeys

Generates a random private key and its corresponding public key on the Stark curve. The keys are cryptographically secure and suitable for production use, with the public key X coordinate serving as the Starknet account address.

## Function Signature

```go
func GetRandomKeys() (privKey, x, y *big.Int, err error)
```

**Source:** [curve.go:L126-L150](https://github.com/NethermindEth/starknet.go/blob/main/curve/curve.go#L126-L150)

## Returns

- `privKey` (*big.Int): The generated private key
- `x` (*big.Int): The x-coordinate of the public key
- `y` (*big.Int): The y-coordinate of the public key
- `err` (error): Error if key generation fails

## Usage Example

```go
package main

import (
	"fmt"
	"log"

	"github.com/NethermindEth/starknet.go/curve"
)

func main() {
	// Generate random keys
	privKey, x, y, err := curve.GetRandomKeys()
	if err != nil {
		log.Fatal("Failed to generate random keys:", err)
	}

	fmt.Println("GetRandomKeys:")
	fmt.Printf("  Private Key: 0x%x\n", privKey)
	fmt.Printf("  Public Key X: 0x%x\n", x)
	fmt.Printf("  Public Key Y: 0x%x\n", y)
	fmt.Println("\nNote: Keys are randomly generated, values will differ on each run")
}
```

## Common Use Cases

- Generate new keypairs for test accounts during development and testing workflows.
- Create temporary session keys for ephemeral authentication or signing operations.
- Generate keys for wallet creation where the private key will be stored securely by the user.
- Use the public key X coordinate as the Starknet account address when deploying new accounts.
- For deriving a public key from an existing private key, use [PrivateKeyToPoint](/docs/curve/functions/private-key-to-point) instead.
