# PrivateKeyToPoint

Generates a public key point from a private key. This function performs scalar multiplication on the Stark curve's base point to derive the corresponding public key coordinates.

## Function Signature

```go
func PrivateKeyToPoint(privKey *big.Int) (x, y *big.Int)
```

**Source:** [curve.go:L152-L168](https://github.com/NethermindEth/starknet.go/blob/main/curve/curve.go#L152-L168)

## Parameters

- `privKey` (*big.Int): The private key

## Returns

- `x` (*big.Int): The x-coordinate of the point on the curve
- `y` (*big.Int): The y-coordinate of the point on the curve

## Usage Example

```go
package main

import (
	"fmt"
	"math/big"

	"github.com/NethermindEth/starknet.go/curve"
)

func main() {
	// Known private key
	privKey := big.NewInt(1234567890123456789)

	// Derive public key
	x, y := curve.PrivateKeyToPoint(privKey)

	fmt.Println("PrivateKeyToPoint:")
	fmt.Printf("  Private Key: 0x%x\n", privKey)
	fmt.Printf("  Public Key X: 0x%x\n", x)
	fmt.Printf("  Public Key Y: 0x%x\n", y)
}
```

## Common Use Cases

- Derive public keys from known private keys when loading existing accounts or wallets.
- Compute the Starknet account address from a stored private key by extracting the X coordinate.
- Verify that a private key corresponds to an expected public key during authentication flows.
- Use in key recovery scenarios where you need to reconstruct the public key from backed-up private keys.
