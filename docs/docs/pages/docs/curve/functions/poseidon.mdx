# Poseidon

Computes the Poseidon hash of two field elements. Poseidon is a cryptographic hash function optimized for SNARK-friendly operations and is more efficient than Pedersen for use in ZK-STARKs.

## Function Signature

```go
func Poseidon(a, b *felt.Felt) *felt.Felt
```

**Source:** [curve.go:L249-L259](https://github.com/NethermindEth/starknet.go/blob/main/curve/curve.go#L249-L259)

## Parameters

- `a` (*felt.Felt): First field element
- `b` (*felt.Felt): Second field element

## Returns

- `*felt.Felt`: The Poseidon hash

## Usage Example

```go
package main

import (
	"fmt"

	"github.com/NethermindEth/juno/core/felt"
	"github.com/NethermindEth/starknet.go/curve"
)

func main() {
	// Create two felt values to hash
	a := new(felt.Felt).SetUint64(123)
	b := new(felt.Felt).SetUint64(456)

	// Compute Poseidon hash
	hash := curve.Poseidon(a, b)

	fmt.Println("Poseidon Hash:")
	fmt.Printf("  Input a: %s\n", a.String())
	fmt.Printf("  Input b: %s\n", b.String())
	fmt.Printf("  Hash: %s\n", hash.String())
}
```

## Common Use Cases

- Use Poseidon when building ZK-STARK proof systems that require efficient cryptographic hashing operations.
- Compute hashes in smart contracts where SNARK-friendly operations provide better performance than traditional hash functions.
- Hash two field elements together when working with Starknet protocols that specifically require Poseidon (rather than Pedersen).
