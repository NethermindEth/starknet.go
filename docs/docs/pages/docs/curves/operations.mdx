# Curve Operations

The curve operations submodule provides functionality for performing operations on curves in Starknet.

## Point Operations

### Point Addition

```go
result := curve.Add(point1, point2)
```

### Point Multiplication

```go
result := curve.Mul(point, scalar)
```

### Point Doubling

```go
result := curve.Double(point)
```

### Point Negation

```go
result := curve.Neg(point)
```

## Field Operations

### Field Addition

```go
result := curve.FieldAdd(a, b)
```

### Field Multiplication

```go
result := curve.FieldMul(a, b)
```

### Field Division

```go
result := curve.FieldDiv(a, b)
```

### Field Inversion

```go
result := curve.FieldInv(a)
```

## Curve Operations

### Curve Order

```go
order := curve.Order()
```

### Curve Generator

```go
generator := curve.Generator()
```

### Curve Is On Curve

```go
isOnCurve := curve.IsOnCurve(point)
```

### Curve Is Valid

```go
isValid := curve.IsValid(point)
```

## Example Usage

```go
// Create a curve
curve := curves.NewStarkCurve()

// Add two points
result := curve.Add(point1, point2)

// Multiply a point by a scalar
result := curve.Mul(point, scalar)

// Double a point
result := curve.Double(point)

// Negate a point
result := curve.Neg(point)

// Add two field elements
result := curve.FieldAdd(a, b)

// Multiply two field elements
result := curve.FieldMul(a, b)

// Divide two field elements
result := curve.FieldDiv(a, b)

// Invert a field element
result := curve.FieldInv(a)

// Get the curve order
order := curve.Order()

// Get the curve generator
generator := curve.Generator()

// Check if a point is on the curve
isOnCurve := curve.IsOnCurve(point)

// Check if a point is valid
isValid := curve.IsValid(point)
```
