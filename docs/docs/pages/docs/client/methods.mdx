# Client Methods

The client methods submodule provides functionality for interacting with Starknet nodes.

## Block Methods

### Get Block by Number

```go
block, err := client.GetBlockByNumber(blockNumber) // [!code focus]
if err != nil { 
    // Handle error
}
```

### Get Block by Hash

```go
block, err := client.GetBlockByHash(blockHash) // [!code focus]
if err != nil {
    // Handle error
}
```

### Get Latest Block

```go
block, err := client.GetLatestBlock() // [!code focus]
if err != nil {
    // Handle error
}
```

## Example Usage

```go
// Get the latest block // [!code focus]
block, err := client.GetLatestBlock() // [!code focus]
if err != nil {
    log.Fatal(err)
}

fmt.Printf("Latest block number: %d\n", block.Number) // [!code focus]
fmt.Printf("Latest block hash: %s\n", block.Hash) // [!code focus]

// Get a transaction // [!code focus]
tx, err := client.GetTransactionByHash("0x123...") // [!code focus]
if err != nil {
    log.Fatal(err)
}

fmt.Printf("Transaction type: %s\n", tx.Type) // [!code focus]
fmt.Printf("Transaction status: %s\n", tx.Status) // [!code focus]

// Call a contract // [!code focus]     
result, err := client.CallContract( // [!code focus]
    "0x456...", // contract address // [!code focus]
    "0x789...", // entry point selector // [!code focus]
    []string{"0xabc..."}, // calldata // [!code focus]
)
if err != nil {
    log.Fatal(err)
}

fmt.Printf("Call result: %s\n", result) // [!code focus]
```
