# Client Package

The client package provides functionality for interacting with Starknet nodes. This package contains various components and utilities for managing and interacting with Starknet nodes.

## Overview

The client package includes:
- Client configuration
- Client methods
- Client examples

## Available Submodules

- [Client Configuration](/docs/client/configuration)
- [Client Methods](/docs/client/methods)
- [Client Examples](/docs/client/examples)

## Getting Started

To use the client package, import it in your Go code:

```go
import "github.com/NethermindEth/starknet.go/client"
```
