# TransactionHashDeployAccountV1

import { Callout } from 'vocs/components'

<Callout type="info">
  **Note**: V1 transactions are deprecated. For new applications, use [TransactionHashDeployAccountV3](./transaction-hash-deploy-account-v3) for V3 transactions.
</Callout>

Calculates the transaction hash for a Deploy Account V1 transaction using the deprecated Pedersen-based hash calculation. This function is maintained for compatibility with historical account deployments.

## Function Signature

```go
func TransactionHashDeployAccountV1(
    txn *rpc.DeployAccountTxnV1,
    contractAddress *felt.Felt,
    chainID *felt.Felt,
) (*felt.Felt, error)
```

**Source:** [hash.go:L676-L713](https://github.com/NethermindEth/starknet.go/blob/main/hash/hash.go#L676-L713)

## Parameters

- `txn` (*rpc.DeployAccountTxnV1): The Deploy Account V1 transaction containing Version, ClassHash, ContractAddressSalt, ConstructorCalldata, MaxFee, and Nonce
- `contractAddress` (*felt.Felt): The precomputed contract address
- `chainID` (*felt.Felt): The Starknet chain ID

## Returns

- `*felt.Felt`: The calculated transaction hash
- `error`: Error if required parameters are missing

## Usage Example

```go
package main

import (
	"context"
	"fmt"
	"log"
	"os"

	"github.com/NethermindEth/juno/core/felt"
	"github.com/NethermindEth/starknet.go/hash"
	"github.com/NethermindEth/starknet.go/rpc"
	"github.com/joho/godotenv"
)

func main() {
	if err := godotenv.Load(); err != nil {
		log.Printf("Warning: .env file not found: %v", err)
	}

	chainID, _ := new(felt.Felt).SetString("0x534e5f5345504f4c4941")
	classHash, _ := new(felt.Felt).SetString("0x01234567890abcdef1234567890abcdef1234567890abcdef1234567890abcd")
	salt := new(felt.Felt).SetUint64(12345)
	maxFee := new(felt.Felt).SetUint64(2500000000000000)
	nonce := new(felt.Felt).SetUint64(0)

	publicKey, _ := new(felt.Felt).SetString("0x01234567890abcdef1234567890abcdef1234567890abcdef1234567890abcd")
	constructorCalldata := []*felt.Felt{publicKey}

	txn := &rpc.DeployAccountTxnV1{
		Type:                rpc.TransactionTypeDeployAccount,
		Version:             rpc.TransactionV1,
		ClassHash:           classHash,
		ContractAddressSalt: salt,
		ConstructorCalldata: constructorCalldata,
		MaxFee:              maxFee,
		Nonce:               nonce,
		Signature:           []*felt.Felt{},
	}

	contractAddress, _ := new(felt.Felt).SetString("0x04718f5a0fc34cc1af16a1cdee98ffb20c31f5cd61d6ab07201858f4287c938d")

	txHash, err := hash.TransactionHashDeployAccountV1(txn, contractAddress, chainID)
	if err != nil {
		log.Fatalf("Failed to calculate transaction hash: %v", err)
	}

	fmt.Printf("Transaction Hash: %s\n", txHash.String())

	if rpcURL := os.Getenv("STARKNET_RPC_URL"); rpcURL != "" {
		verifyTransaction(txHash, rpcURL)
	}
}

func verifyTransaction(txHash *felt.Felt, rpcURL string) {
	client, err := rpc.NewProvider(context.Background(), rpcURL)
	if err != nil {
		log.Printf("Warning: Could not connect to RPC: %v", err)
		return
	}

	ctx := context.Background()
	tx, err := client.TransactionByHash(ctx, txHash)

	if err == nil {
		fmt.Printf("\nVerification: FOUND on-chain\n")
		fmt.Printf("Type: %T\n", tx)
	} else {
		fmt.Printf("\nVerification: NOT FOUND\n")
	}
}
```

## Common Use Cases

This function is commonly used for:

- Processing historical account deployment transactions when building blockchain explorers that need to verify legacy account creation hashes.
- Implementing account recovery tools that analyze historical deployment transactions to reconstruct account creation patterns and ownership chains.
- Building migration utilities that help users transition from V1 account deployments to V3 by analyzing and verifying legacy deployment transaction data.
- Creating audit systems that verify the integrity of historical account deployment records by independently computing and validating transaction hashes.
- Developing testing frameworks that simulate legacy account deployment scenarios to ensure backward compatibility in wallet and account management systems.
