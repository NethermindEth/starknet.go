# GetSupportedTokens

Retrieves the list of tokens supported by the paymaster for fee payment.

## Method Signature

```go
func (pm *Paymaster) GetSupportedTokens(ctx context.Context) ([]TokenData, error)
```

## Parameters

- `ctx` (context.Context): Context for the operation

## Returns

- `[]TokenData`: Array of supported token information
- `error`: Error if request fails

## TokenData Structure

```go
type TokenData struct {
    TokenAddress  *felt.Felt
    TokenSymbol   string
    TokenDecimals uint8
}
```

## Description

GetSupportedTokens returns all tokens that can be used to pay transaction fees through this paymaster. Each token includes its address, symbol, and decimal precision.

## Usage Example

```go
tokens, err := pm.GetSupportedTokens(ctx)
if err != nil {
    log.Fatal(err)
}

for _, token := range tokens {
    fmt.Printf("Token: %s (%s)\n", token.TokenSymbol, token.TokenAddress)
    fmt.Printf("  Decimals: %d\n", token.TokenDecimals)
}
```

## Use Cases

### 1. Display Fee Options to User
```go
tokens, _ := pm.GetSupportedTokens(ctx)
for i, token := range tokens {
    fmt.Printf("%d. Pay with %s\n", i+1, token.TokenSymbol)
}
```

### 2. Validate Token Selection
```go
tokens, _ := pm.GetSupportedTokens(ctx)
isSupported := false
for _, token := range tokens {
    if token.TokenAddress.Equal(selectedToken) {
        isSupported = true
        break
    }
}
```

### 3. Auto-select Best Token
```go
tokens, _ := pm.GetSupportedTokens(ctx)
// Select token user has balance in
for _, token := range tokens {
    balance, _ := getTokenBalance(userAddr, token.TokenAddress)
    if balance.Cmp(requiredAmount) >= 0 {
        return token
    }
}
```

## Notes

- Token list may change as paymaster adds/removes support
- Check regularly if caching token information
- Decimals needed for amount formatting
