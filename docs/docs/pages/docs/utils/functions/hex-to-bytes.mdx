# HexToBytes

Converts a hexadecimal string to a byte slice.

## Function Signature

```go
func HexToBytes(hexString string) ([]byte, error)
```

## Parameters

- `hexString` - The hexadecimal string to be converted (with or without "0x" prefix)

## Returns

- `[]byte` - The converted byte slice
- `error` - An error if the hex string is invalid

## Description

HexToBytes converts a hexadecimal string representation to a byte array. The function accepts hex strings with or without the "0x" prefix and returns the raw byte representation. This is commonly used when working with Starknet addresses, transaction hashes, or any hex-encoded data that needs to be processed as bytes.

## Usage Example

```go
package main

import (
	"fmt"
	"github.com/NethermindEth/starknet.go/utils"
)

func main() {
	// Convert hex string to bytes
	hexStr := "0x48656c6c6f20576f726c64"
	bytes, err := utils.HexToBytes(hexStr)
	if err != nil {
		fmt.Printf("Error: %v\n", err)
		return
	}

	fmt.Printf("Hex: %s\n", hexStr)
	fmt.Printf("Bytes: %v\n", bytes)
	fmt.Printf("String: %s\n", string(bytes))
	// Output:
	// Hex: 0x48656c6c6f20576f726c64
	// Bytes: [72 101 108 108 111 32 87 111 114 108 100]
	// String: Hello World
}
```

## Expected Output

```
HexToBytes:
  Input hex: 0x48656c6c6f
  Bytes: [72 101 108 108 111]
  ASCII: Hello
```

## Error Handling

```go
// Invalid hex string
invalidHex := "0xGGHH"
_, err := utils.HexToBytes(invalidHex)
if err != nil {
	fmt.Printf("Invalid hex string: %v\n", err)
}
```

## Common Use Cases

1. **Address Processing**: Convert hex address strings to bytes for hashing
2. **Transaction Data**: Decode hex-encoded calldata or signatures
3. **Hash Operations**: Convert hex hashes to bytes for verification
4. **Binary Protocol Handling**: Decode hex-encoded network data
