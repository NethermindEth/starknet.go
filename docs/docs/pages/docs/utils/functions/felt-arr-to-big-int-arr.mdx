# FeltArrToBigIntArr

Converts an array of Felt objects to an array of big.Int objects.

## Function Signature

```go
func FeltArrToBigIntArr(f []*felt.Felt) []*big.Int
```

## Parameters

- `f` - The array of `*felt.Felt` objects to convert

## Returns

- `[]*big.Int` - The array of converted `*big.Int` objects

## Description

FeltArrToBigIntArr converts an entire array of Starknet Felt values to an array of `*big.Int` values. This is useful when you need to perform mathematical operations or conversions that require `big.Int` types rather than Felt types, or when interfacing with code that expects big integers.

## Usage Example

```go
package main

import (
	"fmt"
	"github.com/NethermindEth/juno/core/felt"
	"github.com/NethermindEth/starknet.go/utils"
)

func main() {
	// Create an array of Felt values
	feltArr := []*felt.Felt{
		new(felt.Felt).SetUint64(100),
		new(felt.Felt).SetUint64(200),
		new(felt.Felt).SetUint64(300),
	}

	// Convert to big.Int array
	bigIntArr := utils.FeltArrToBigIntArr(feltArr)

	fmt.Println("Felt Array:")
	for i, f := range feltArr {
		fmt.Printf("  [%d]: %s\n", i, f.String())
	}

	fmt.Println("\nBig Int Array:")
	for i, b := range bigIntArr {
		fmt.Printf("  [%d]: %s\n", i, b.String())
	}
	// Output:
	// Felt Array:
	//   [0]: 100
	//   [1]: 200
	//   [2]: 300
	//
	// Big Int Array:
	//   [0]: 100
	//   [1]: 200
	//   [2]: 300
}
```

## Expected Output

```
FeltArrToBigIntArr:
  Input Felt array: [123, 456, 789]
  Output big.Int array: [123, 456, 789]
```

## Common Use Cases

1. **Legacy Code Integration**: Interface with existing code that uses `big.Int`
2. **Mathematical Operations**: Perform operations not available on Felt types
3. **Data Export**: Convert Starknet data to formats that use big integers
4. **Cross-Package Operations**: Work with packages that expect `big.Int` arrays

## Performance Notes

- Conversion is done element-by-element
- Original Felt array is not modified
- Returns a new array of big.Int pointers
