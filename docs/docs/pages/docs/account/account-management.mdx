# Account Management

The account management submodule provides core functionality for creating and managing Starknet accounts.

## Account Structure

An account in Starknet.go is represented by the `Account` struct, which contains:

- Public key
- Private key
- Address
- Chain ID
- Provider interface

## Creating an Account

To create a new account:

```go 
 // [!code focus] 
account, err := account.NewAccount(privateKey, address, chainID, provider) // [!code focus]
if err != nil {
    // Handle error
}
```

## Account Methods

### Sign

Sign a transaction or message:

```go
// [!code focus]
signature, err := account.Sign(message) // [!code focus]
if err != nil {
    // Handle error
}
```

### Verify

Verify a signature:

```go
isValid := account.Verify(message, signature)
```

### GetAddress

Get the account's address:

```go
address := account.GetAddress()
```

## Example Usage

```go
// Create a new account
privateKey := "0x123..." // Your private key
address := "0x456..."    // Your account address
chainID := "SN_MAIN"     // Starknet chain ID
provider := // Your provider implementation

// [!code focus] //
account, err := account.NewAccount(privateKey, address, chainID, provider) // [!code focus]
if err != nil {
    log.Fatal(err)
}

// Sign a message
message := []byte("Hello, Starknet!") // [!code focus]
signature, err := account.Sign(message) // [!code focus]
if err != nil {
    log.Fatal(err)
}

// Verify the signature
isValid := account.Verify(message, signature) // [!code focus]
fmt.Printf("Signature valid: %v\n", isValid) // [!code focus]
```
