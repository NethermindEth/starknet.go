# Utilities Package

The Utilities package provides common utility functions and helpers for working with Starknet. This package contains various components and utilities for common operations.

## Overview

The Utilities package includes:
- Common utilities
- Type utilities
- Conversion utilities

## Available Submodules

- [Common Utilities](./common)
- [Type Utilities](./types)

## Getting Started

To use the Utilities package, import it in your Go code:

```go
import "github.com/NethermindEth/starknet.go/utils"
```

## Version

This documentation is based on version v0.8.1 of the Starknet.go library.
