# Curves Package

The Curves package provides functionality for working with elliptic curves in Starknet. This package contains various components and utilities for curve operations.

## Overview

The Curves package includes:
- Curve types
- Curve operations
- Curve utilities

## Available Submodules

- [Curve Types](./types)
- [Curve Operations](./operations)

## Getting Started

To use the Curves package, import it in your Go code:

```go
import "github.com/NethermindEth/starknet.go/curves"
```
