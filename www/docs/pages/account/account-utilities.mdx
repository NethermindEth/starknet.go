# Account Utilities

The account utilities submodule provides helper functions and utilities for working with Starknet accounts.

## Utility Functions

### Address Validation

```go
isValid := account.ValidateAddress(address)
```

### Public Key Generation

```go
publicKey := account.GeneratePublicKey(privateKey)
```

### Address Generation

```go
address := account.GenerateAddress(publicKey)
```

### Chain ID Validation

```go
isValid := account.ValidateChainID(chainID)
```

## Example Usage

```go
// Validate an address
address := "0x123..."
isValid := account.ValidateAddress(address)
fmt.Printf("Address valid: %v\n", isValid)

// Generate public key from private key
privateKey := "0x456..."
publicKey := account.GeneratePublicKey(privateKey)
fmt.Printf("Public key: %s\n", publicKey)

// Generate address from public key
address = account.GenerateAddress(publicKey)
fmt.Printf("Generated address: %s\n", address)

// Validate chain ID
chainID := "SN_MAIN"
isValid = account.ValidateChainID(chainID)
fmt.Printf("Chain ID valid: %v\n", isValid)
```

## Common Utilities

### Hex Encoding/Decoding

```go
// Encode bytes to hex
hex := account.EncodeHex(bytes)

// Decode hex to bytes
bytes, err := account.DecodeHex(hex)
```

### Field Element Conversion

```go
// Convert to field element
fe := account.ToFieldElement(value)

// Convert from field element
value := account.FromFieldElement(fe)
```

### Error Handling

```go
// Check for specific errors
if err != nil {
    if account.IsInvalidAddressError(err) {
        // Handle invalid address error
    } else if account.IsInvalidSignatureError(err) {
        // Handle invalid signature error
    }
}
```
