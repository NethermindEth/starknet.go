# Transaction Handling

The transaction handling submodule provides functionality for creating, signing, and sending transactions on Starknet.

## Transaction Types

The package supports various transaction types:

- Invoke transactions
- Deploy transactions
- Declare transactions

## Creating Transactions

### Invoke Transaction

```go
tx := account.NewInvokeTransaction(
    contractAddress,
    entryPointSelector,
    calldata,
    maxFee,
    version,
    nonce,
)
```

### Deploy Transaction

```go
tx := account.NewDeployTransaction(
    contractClassHash,
    constructorCalldata,
    salt,
    version,
)
```

## Transaction Methods

### Sign Transaction

```go
signedTx, err := account.SignTransaction(tx)
if err != nil {
    // Handle error
}
```

### Send Transaction

```go
txHash, err := account.SendTransaction(signedTx)
if err != nil {
    // Handle error
}
```

### Wait for Transaction

```go
receipt, err := account.WaitForTransaction(txHash)
if err != nil {
    // Handle error
}
```

## Example Usage

```go
// Create an invoke transaction
tx := account.NewInvokeTransaction(
    "0x123...", // contract address
    "0x456...", // entry point selector
    []string{"0x789..."}, // calldata
    "1000000000000000", // max fee
    "0x1", // version
    "0x1", // nonce
)

// Sign the transaction
signedTx, err := account.SignTransaction(tx)
if err != nil {
    log.Fatal(err)
}

// Send the transaction
txHash, err := account.SendTransaction(signedTx)
if err != nil {
    log.Fatal(err)
}

// Wait for transaction confirmation
receipt, err := account.WaitForTransaction(txHash)
if err != nil {
    log.Fatal(err)
}

fmt.Printf("Transaction confirmed with status: %s\n", receipt.Status)
```
