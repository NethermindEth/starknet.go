# RPC Package

The RPC package provides functionality for interacting with Starknet nodes using JSON-RPC. This package contains various components and utilities for making RPC calls to Starknet nodes.

## Overview

The RPC package includes:
- RPC methods
- RPC examples
- RPC utilities

## Available Submodules

- [RPC Methods](/docs/rpc/methods)
- [RPC Examples](/docs/rpc/examples)

## Getting Started

To use the RPC package, import it in your Go code:

```go
import "github.com/NethermindEth/starknet.go/rpc"
```
