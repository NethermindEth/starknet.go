# Get State Update

Returns the information about the result of executing the requested block. 

## Parameters

- `ctx` - The context of the request.
- `blockID` - The block identifier. Can be:
  - `BlockHash` - The hash of the block.
  - `BlockNumber` - The number of the block.
  - `"latest"` for the latest block
  - `"pending"` for the pending block

## Returns

- `StateUpdateOutput` - The state update object containing:
  - `NewRoot` - The new root.
  - `OldRoot` - The old root.
  - `StateDiff` - The state diff.
  - `BlockHash` - The hash of the block.
  - `BlockNumber` - The number of the block.
- `error` - An error if any occurred.

## Example

```go
package main

import (
	"context"
	"fmt"
	"log"
	"github.com/NethermindEth/starknet.go/rpc"

	"encoding/json"
)

func main() {
	rpcUrl := "https://starknet-mainnet.public.blastapi.io/rpc/v0_8"
	provider, err := rpc.NewProvider(rpcUrl)
	if err != nil {
		log.Fatal(err)
	}

	result, err := provider.StateUpdate(context.Background(), rpc.BlockID{Tag: "latest"})
	if err != nil {
		log.Fatal(err)
	}
	formartedResult, err := json.MarshalIndent(result, "", "  ")
	if err != nil {
		log.Fatal(err)
	}
	fmt.Println("State Update Output", string(formartedResult))
}
```
