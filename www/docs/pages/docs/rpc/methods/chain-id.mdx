# Chain ID

Returns the chain ID for transaction replay protection.

## Parameters

- `ctx` - The context of the request.

## Returns

- `result` - The chain ID as a Felt.
- `error` - An error if any occurred.

## Example

```go
package main

import (
	"context"
	"fmt"
	"log"
	"github.com/NethermindEth/starknet.go/rpc"
)

func main() {
	rpcUrl := "https://starknet-mainnet.public.blastapi.io/rpc/v0_8"
	provider, err := rpc.NewProvider(rpcUrl)
	if err != nil {
		log.Fatal(err)
	}

	result, err := provider.ChainID(context.Background())
	if err != nil {
		log.Fatal(err)
	}
	fmt.Println("Chain ID:", result)
}
```

## Output
```json
Chain ID: SN_MAIN
```

## Error Cases

None
