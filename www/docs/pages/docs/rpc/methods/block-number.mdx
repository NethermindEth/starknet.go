# Block Number

Returns the latest block number.

## Parameters

None

## Returns

Returns the latest block number as an integer.

## Example

```go
package main

import (
	"context"
	"fmt"
	"log"
	"github.com/NethermindEth/starknet.go/rpc"
)

func main() {
	rpcUrl := "https://free-rpc.nethermind.io/mainnet-juno/"
	client, err := rpc.NewClient(rpcUrl)
	if err != nil {
		log.Fatal(err)
	}

	provider := rpc.NewProvider(client)
	
	result, err := provider.BlockNumber(context.Background())
	if err != nil {
		log.Fatal(err)
	}
	fmt.Println("Block Number:", result)
}
```
