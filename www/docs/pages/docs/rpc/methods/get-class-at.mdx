# Get Class At

Returns the contract class definition at the given address.

## Parameters

- `contract_address` - The address of the contract
- `block_id` - The block identifier. Can be:
  - Block hash
  - Block number
  - "latest" for the latest block
  - "pending" for the pending block

## Returns

Returns a contract class object containing:
- Program
- Entry points
- ABI
- Other class-specific data

## Example

```go
package main

import (
	"context"
	"fmt"
	"log"
	"github.com/NethermindEth/starknet.go/rpc"
	"github.com/NethermindEth/starknet.go/utils"
)

func main() {
	rpcUrl := "https://free-rpc.nethermind.io/mainnet-juno/"
	client, err := rpc.NewClient(rpcUrl)
	if err != nil {
		log.Fatal(err)
	}

	provider := rpc.NewProvider(client)
	
	// Convert string address to Felt
	contractAddress, _ := utils.HexToFelt("0x123...")
	
	result, err := provider.ClassAt(context.Background(), rpc.BlockID{Tag: "latest"}, contractAddress)
	if err != nil {
		log.Fatal(err)
	}
	fmt.Println("Class:", result)
}
```

## Error Cases

- `CONTRACT_NOT_FOUND` - If the contract does not exist
- `INVALID_BLOCK_ID` - If the block identifier is invalid
- `INVALID_CONTRACT_ADDRESS` - If the contract address is invalid
