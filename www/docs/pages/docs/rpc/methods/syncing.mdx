# Syncing

Returns the syncing status of the node.

## Parameters

- `ctx` - The context of the request.

## Returns
- `*SyncStatus` - A pointer to a SyncingStatus object.

## Example

```go
package main

import (
	"context"
	"fmt"
	"log"
	"github.com/NethermindEth/starknet.go/rpc"

	"encoding/json"
)

func main() {
	rpcUrl := "https://free-rpc.nethermind.io/mainnet-juno/"
	provider, err := rpc.NewProvider(rpcUrl)
	if err != nil {
		log.Fatal(err)
	}

	result, err := provider.Syncing(context.Background())
	if err != nil {
		log.Fatal(err)
	}

	resultJson, err := json.MarshalIndent(result, "", "  ")
	if err != nil {
		log.Fatal(err)
	}
	fmt.Println("SyncStatus:", resultJson)
}
```

## Output 
```json
SyncStatus: [102 97 108 115 101]
```

## Error Cases

None
