# Syncing

Returns the syncing status of the node.

## Parameters

None

## Returns

Returns a syncing status object containing:
- Starting block number
- Current block number
- Highest block number
- Other syncing-related data

## Example

```go
package main

import (
	"context"
	"fmt"
	"log"
	"github.com/NethermindEth/starknet.go/rpc"
)

func main() {
	rpcUrl := "https://free-rpc.nethermind.io/mainnet-juno/"
	client, err := rpc.NewClient(rpcUrl)
	if err != nil {
		log.Fatal(err)
	}

	provider := rpc.NewProvider(client)
	
	result, err := provider.Syncing(context.Background())
	if err != nil {
		log.Fatal(err)
	}
	fmt.Println("Syncing Status:", result)
}
```
