# Estimate Message Fee

Estimates the fee for a message.

## Parameters

- `message` - The message request object containing:
  - From address
  - To address
  - Entry point selector
  - Payload
  - Other message-specific data
- `block_id` - The block identifier. Can be:
  - Block hash
  - Block number
  - "latest" for the latest block
  - "pending" for the pending block

## Returns

Returns a fee estimate object containing:
- Gas price
- Gas usage
- Overall fee
- Other fee-related data

## Example

```go
package main

import (
	"context"
	"fmt"
	"log"
	"github.com/NethermindEth/starknet.go/rpc"
	"github.com/NethermindEth/starknet.go/utils"
)

func main() {
	rpcUrl := "https://free-rpc.nethermind.io/mainnet-juno/"
	client, err := rpc.NewClient(rpcUrl)
	if err != nil {
		log.Fatal(err)
	}

	provider := rpc.NewProvider(client)
	
	// Convert string addresses to Felt
	fromAddress, _ := utils.HexToFelt("0x123...")
	toAddress, _ := utils.HexToFelt("0x456...")
	entryPointSelector, _ := utils.HexToFelt("0x789...")
	
	// Create message request
	message := rpc.MsgFromL1{
		FromAddress: fromAddress,
		ToAddress: toAddress,
		EntryPointSelector: entryPointSelector,
		Payload: []*felt.Felt{},
	}
	
	result, err := provider.EstimateMessageFee(context.Background(), message, rpc.BlockID{Tag: "latest"})
	if err != nil {
		log.Fatal(err)
	}
	fmt.Println("Message Fee Estimate:", result)
}
```

## Error Cases

- `INVALID_BLOCK_ID` - If the block identifier is invalid
- `INVALID_MESSAGE` - If the message is invalid
- `CONTRACT_NOT_FOUND` - If the contract does not exist
- `INVALID_CONTRACT_ADDRESS` - If the contract address is invalid
